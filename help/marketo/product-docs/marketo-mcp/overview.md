# Document de présentation MCP {#overview}

TEXTE D&#39;UN PÈRE :

Le serveur MCP public Marketo Engage est un service basé sur le cloud qui rend les API REST Marketo accessibles aux agents d’IA en tant qu’outils structurés. Basée sur le protocole MCP (Model Context Protocol), elle fournit une interface normalisée pilotée par des schémas qui permet aux systèmes d’IA d’interagir avec Marketo de manière sécurisée et efficace.

Au lieu d’intégrer manuellement les points d’entrée Marketo dans chaque application, le serveur MCP fournit une passerelle native unique permettant aux agents compatibles avec MCP de découvrir les opérations disponibles, de comprendre les paramètres requis et d’exécuter des actions via un protocole cohérent.

Il existe un grand nombre d’outils disponibles qui comprennent des API REST publiques et des API internes, qui exécutent des fonctions principales de Marketo comme dans l’interface utilisateur, mais par le biais d’appels d’API. D’autres outils seront ajoutés au fur et à mesure de leur développement. &lt;---- BESOIN D&#39;ÉCLAIRCISSEMENTS

Le serveur MCP est proposé en libre-service et peut être configuré via le framework d’E/S d’Adobe en créant un point d’entrée et en authentifiant la connexion. Pour plus d’informations et d’informations sur la configuration, consultez le site d’Adobe Developer (ajoutez un lien ici). CELUI-CI ? https://developer.adobe.com/marketo-apis/

Les outils Campaign affichés sous la forme d’API SOAP ont été convertis en API REST et ont été ajoutés.

Une fois configuré, pour afficher la liste des outils disponibles, suivez ces instructions ici (lien hypertexte).

REMARQUE : les outils qui ne sont pas disponibles sont « Désactiver la campagne » et « Supprimer la liste statique » (à déterminer).
