# Erreur lors de l&#39;envoi à partir de l&#39;Application web {#error-when-sending-from-the-web-application}

En règle générale, lorsqu’une erreur est générée à partir de l’[application Web](https://toutapp.com/login), cela est dû à un problème de mise en forme de votre courrier électronique ou aux champs dynamiques. Assurez-vous que tout est correctement renseigné dans la fenêtre de composition, puis réessayez d’envoyer.
