# Ajouter du contenu {#adding-content}

Pour ajouter du contenu pouvant faire l’objet d’un suivi à vos messages électroniques, ouvrez une nouvelle fenêtre Composer. Cliquez sur **Contenu**.

Si vous avez précédemment ajouté du contenu, vous verrez ces fichiers. Dans le cas contraire, cliquez sur **Télécharger** pour ajouter du contenu à partir de votre ordinateur.

>[!NOTE]
>
>Le fichier sera placé à l&#39;endroit où se trouve votre curseur dans le courriel, et vous devez cliquer sur &quot;Tout&quot; lors de l&#39;envoi du courriel afin de suivre le contenu de votre flux en direct.
