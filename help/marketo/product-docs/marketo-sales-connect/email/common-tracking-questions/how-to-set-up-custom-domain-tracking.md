# Configuration du suivi de domaine personnalisé {#how-to-set-up-custom-domain-tracking}

Nous vous offrons la possibilité de configurer le suivi personnalisé des domaines pour vos liens. Ainsi, lorsqu’un utilisateur survole un lien, celui-ci lit &quot;go.votreentreprise.com&quot; au lieu de &quot;go.toutapp.com&quot;.

Pour configurer cette configuration, votre administrateur informatique doit suivre les étapes répertoriées dans notre [page paramètres](https://toutapp.com/custom_tracking_domain).

Une fois cette configuration effectuée, nous placerons sur la liste autorisée tout lien contenu dans votre email qui apparaîtra sous la forme &quot;go.yourcompany.com&quot;.
