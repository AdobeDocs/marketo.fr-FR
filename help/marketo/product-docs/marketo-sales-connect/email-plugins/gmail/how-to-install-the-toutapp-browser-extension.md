# Comment installer l’extension de navigateur ToutApp {#how-to-install-the-toutapp-browser-extension}

1. Cliquez sur **Paramètres** sur le [application web](https://toutapp.com/next#settings).

1. Sous Mon compte, cliquez sur **Paramètres de messagerie électronique**.

1. Cliquez sur **Modules externes**.

1. Sous les applications Gmail et Google, cliquez sur **Installer**.
