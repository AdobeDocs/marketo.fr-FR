# Comment installer l’extension de navigateur ToutApp {#how-to-install-the-toutapp-browser-extension}

1. Cliquez sur **Settings** sur l’ [application web](https://toutapp.com/next#settings).

1. Sous Mon compte, cliquez sur **Paramètres de courrier électronique**.

1. Cliquez sur **Plugins**.

1. Sous les applications Gmail et Google, cliquez sur **Installer**.
