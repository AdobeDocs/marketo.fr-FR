# SEO - Afficher les résultats des mots-clés dans un moteur de recherche {#seo-view-keyword-results-in-search-engine}

Si vous souhaitez voir à quoi ressemble votre contenu dans un moteur de recherche, nous avons ajouté un petit raccourci pratique.

1. Accédez à la section **[!UICONTROL Mots-clés]**.

   ![](assets/image2014-9-18-13-3a33-3a58.png)

1. Pointez sur un mot-clé. Cliquez sur **[!UICONTROL Afficher les résultats]**.

   ![](assets/image2014-9-18-13-3a34-3a2.png)

1. Choisissez le moteur de recherche dans lequel afficher vos résultats.

   ![](assets/image2014-9-18-13-3a34-3a16.png)

Vous devriez maintenant voir les résultats de la recherche pour votre mot-clé dans une nouvelle fenêtre.

![](assets/image2014-9-18-13-3a34-3a24.png)
