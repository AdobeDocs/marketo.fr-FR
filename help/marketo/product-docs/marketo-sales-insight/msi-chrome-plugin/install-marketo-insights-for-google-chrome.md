# Installation de Marketo Insights pour Google Chrome {#install-marketo-insights-for-google-chrome}

Suivez ces étapes pour commencer à utiliser le puissant module complémentaire Chrome.

>[!NOTE]
>
>Vous n’avez pas besoin d’être un administrateur Marketo pour installer l’extension.

1. Installez le [Ajout de Marketo Insights for Google Chrome dans l’extension](https://chrome.google.com/webstore/detail/marketo-for-google-mail/jjkfbhajlmoeegbjgjipliamplidmbjb) à partir de Chrome Web Store.

   ![](assets/image2015-10-5-10-3a24-3a7.png)

1. Cliquez avec le bouton droit sur le logo Marketo dans Chrome, puis sélectionnez **Options**.

   ![](assets/two.png)

1. Saisissez votre **Code reg**, **Adresse électronique**, **Prénom**, et **Nom**. Cliquez sur **Vérification et enregistrement**.

   ![](assets/three.png)

   >[!CAUTION]
   >
   >Veillez à utiliser **Principal compte de messagerie** lors de l’enregistrement, car nous ne prenons pas en charge l’utilisation d’alias pour ce module externe.

   >[!NOTE]
   >
   >Le code Reg figure dans l’e-mail envoyé après un administrateur Marketo. [vous émet une licence de module complémentaire de messagerie Marketo](/help/marketo/product-docs/marketo-sales-insight/msi-outlook-plugin/issue-a-marketo-email-add-in-license.md) [.](https://docs.marketo.com/pages/viewpage.action?pageid=7510848) **Il expire après 14 jours**.

1. Cliquez sur **Autoriser** pour autoriser l’accès hors ligne.

   ![](assets/image2015-10-5-10-3a34-3a1.png)
