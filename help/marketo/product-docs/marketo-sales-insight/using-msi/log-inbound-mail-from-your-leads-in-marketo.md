# Enregistrer le courrier entrant provenant de vos leads dans Marketo {#log-inbound-mail-from-your-leads-in-marketo}

Vous pouvez enregistrer les réponses de vos prospects directement dans [!DNL Outlook] avec le complément d’e-mail Marketo.

## À partir de l’application [!DNL Outlook] principale {#from-the-main-outlook-application}

1. Sélectionnez l’e-mail à enregistrer et cliquez sur **[!UICONTROL Enregistrer avec Marketo]**.

![](assets/image2014-9-23-17-3a12-3a44.png)

>[!TIP]
>
>Vous pouvez également cliquer avec le bouton droit de la souris sur un message, puis cliquer sur **[!UICONTROL Se connecter à Marketo]**.

Vous devriez voir une confirmation.

![](assets/image2014-9-23-17-3a13-3a39.png)

## À partir de l’e-mail lui-même {#from-the-email-itself}

Si vous avez ouvert l’e-mail, vous pouvez simplement cliquer sur le bouton **[!UICONTROL Se connecter avec Marketo]** à partir de là.

![](assets/image2014-9-23-17-3a14-3a14.png)

Vous devriez voir la même confirmation que l’autre méthode.

![](assets/image2014-9-23-17-3a14-3a29.png)

Enregistrez les réponses de votre prospect pour les ajouter à leur historique dans Marketo.
