# Exportation de métadonnées d’objet personnalisé {#custom-object-metadata-export}

Si vous utilisez notre API SOAP ou notre API Munchkin, le Schéma de métadonnées d’objet personnalisé peut être exporté. Voici comment l&#39;obtenir.

1. Dans Mon Marketo, cliquez sur **Admin**.

   ![](assets/custom-object-metadata-export-1.png)

1. Cliquez sur **Marketo Custom Objects**.

   ![](assets/custom-object-metadata-export-2.png)

1. Sélectionnez l’objet personnalisé Marketo à exporter.

   ![](assets/custom-object-metadata-export-3.png)

1. Cliquez sur la liste déroulante **Actions d’objet personnalisé** et sélectionnez **Exporter l’objet**.

   ![](assets/custom-object-metadata-export-4.png)

>[!NOTE]
>
>L&#39;objet personnalisé doit être dans l&#39;état Approuvé pour être exporté.

Vous disposez maintenant d’une feuille de calcul avec le Schéma de l’objet personnalisé, sur trois onglets.

![](assets/custom-object-metadata-export-5.png)

![](assets/custom-object-metadata-export-6.png)
