# Configuration de l’administrateur de produit {#product-admin-setup}

1. Une fois que vous avez été invité par un administrateur système, vous recevrez un e-mail de bienvenue. Dans cet email, cliquez sur **[!UICONTROL Commencer]**.

   ![](assets/admin-setup-7.png)

1. Si vous avez déjà accédé à une application avec une Adobe ID, vous serez directement dirigé vers Adobe Admin Console. Sinon, [configurez votre Adobe ID](https://helpx.adobe.com/fr/manage-account/using/create-update-adobe-id.html){target="_blank"}.

   ![](assets/admin-setup-8.png)

C&#39;est aussi simple que ça ! Les administrateurs de produit sont principalement chargés d’ajouter des utilisateurs. [Découvrez comment le faire ici](/help/marketo/product-docs/administration/marketo-with-adobe-identity/add-or-remove-a-user.md#add-a-user){target="_blank"}.
