# Configuration minimale requise {#minimum-system-requirements}

Si vous vous demandez ce dont vous avez besoin pour vous connecter et utiliser Marketo Engage, voici les informations que vous recherchez.

>[!NOTE]
>
>iOS et Android ne sont pas encore pris en charge.

* Ordinateur ou portable exécutant [!DNL Windows], Mac OS X (ou version ultérieure) ou Linux
* Clavier et souris
* Moniteur avec une résolution minimale de 1280 x 768
* Navigateur à jour, voir [Navigateurs pris en charge](/help/marketo/product-docs/administration/setup-administration/supported-browsers.md){target="_blank"} pour plus d’informations
* Connexion Internet

>[!NOTE]
>
>Si votre équipe informatique restreint l’accès web à l’aide d’un, demandez-lui d’ajouter des `*.marketo.com`, des `*.adobe.com` et des `*.adobe.net` pour autoriser toutes les ressources et tous les sockets web de Marketo Engage.
