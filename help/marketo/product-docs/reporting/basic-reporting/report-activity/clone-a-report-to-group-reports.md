# Clonage d’un rapport pour regrouper des rapports {#clone-a-report-to-group-reports}

Tout rapport dans **Mes rapports **n’est visible que par vous. Que se passe-t-il si vous souhaitez rendre le rapport accessible à tous les autres ? Voici comment vous pouvez faire cela :

1. Dans le **Analytics** , cliquez sur la zone de votre choix. [rapport enregistré](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/save-a-report.md).

   ![](assets/image2015-4-21-11-3a25-3a54.png)

   >[!NOTE]
   >
   >Vous pouvez également cloner un rapport enregistré en **Rapports sur les groupes** en faisant glisser le rapport dans le dossier .

1. Cliquez sur **Actions de rapport** et sélectionnez **Cloner le rapport**.

   ![](assets/image2015-4-21-11-3a29-3a32.png)

1. Cliquez sur le bouton **Cloner vers** et sélectionnez votre groupe. Sélectionnez la **Dossier**.

   ![](assets/image2015-4-21-11-3a32-3a0.png)

1. **Nom** le rapport, puis cliquez sur **Cloner**.

   ![](assets/image2015-4-21-11-3a33-3a11.png)

   Super ! Votre rapport apparaît désormais dans **Rapports sur les groupes**.

   ![](assets/image2015-4-21-11-3a37-3a25.png)
