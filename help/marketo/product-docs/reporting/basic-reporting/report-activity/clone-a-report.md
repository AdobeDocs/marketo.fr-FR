# Clonage d’un rapport {#clone-a-report}

Créez un double de votre rapport pour le personnaliser davantage sans modifier l’original.

1. Accédez à **Activités marketing** (ou **Analytics**).

   ![](assets/image2014-9-16-14-3a23-3a46.png)

1. Sélectionnez votre rapport. Sélectionner **Actions de rapport** > **Clonage d’un rapport**.

   ![](assets/image2014-9-16-14-3a23-3a53.png)

   >[!TIP]
   >
   >Vous pouvez également cloner un rapport en cliquant avec le bouton droit de la souris dans l’arborescence.

1. Nommez votre clone de rapport.

   ![](assets/image2014-9-16-14-3a23-3a57.png)

   C&#39;est tout. Vous êtes prêt à personnaliser votre clone !
