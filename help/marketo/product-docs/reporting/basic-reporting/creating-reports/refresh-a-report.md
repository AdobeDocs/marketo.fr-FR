# Actualiser un rapport {#refresh-a-report}

Après avoir consulté un rapport, Marketo le stocke dans votre base de données afin qu’il se charge rapidement lors de la consultation suivante. Suite à votre première consultation, les rapports sont automatiquement actualisés toutes les 24 heures afin d’être toujours à jour. Vous pouvez toutefois les actualiser manuellement à tout moment.

1. Pour savoir quand votre rapport a été mis à jour pour la dernière fois, pointez sur l’icône de flèche en forme de cercle dans le coin inférieur gauche.

   ![](assets/one.png)

1. Cette icône de flèche en forme de cercle est le bouton d’actualisation. Il suffit de cliquer dessus pour obtenir les derniers résultats.

   ![](assets/two.png)

1. Vous pouvez également actualiser le rapport en cliquant sur le menu **[!UICONTROL Actions du rapport]** et en sélectionnant **[!UICONTROL Actualiser le rapport]**.

   ![](assets/three.png)

   Voilà !
