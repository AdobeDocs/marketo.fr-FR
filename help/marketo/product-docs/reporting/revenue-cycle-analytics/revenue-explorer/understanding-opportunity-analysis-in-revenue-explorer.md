# Présentation de l’analyse des opportunités dans l’Explorateur des recettes {#understanding-opportunity-analysis-in-revenue-explorer}

L’ analyse des opportunités dans l’Explorateur des recettes vous permet d’examiner vos opportunités à un niveau plus détaillé. Divisez et dressez les données en fonction d’attributs de prospect/société, tels que la source de prospect, le secteur industriel ou la zone géographique. Analysez la création d’opportunités et la fermeture en fonction du nom, de l’étape ou de la probabilité. Découvrez les contributions marketing au pipeline.

## Exemple d&#39;analyse {#example-analysis}

Voici quelques rapports que vous pouvez créer dans la zone Analyse des opportunités .

1. Influence marketing sur les opportunités créées

   Quel pourcentage du pipeline de votre entreprise a été influencé par vos programmes marketing ? Ce rapport vous donne la réponse. Les graphiques circulaires suivants montrent le pourcentage du nombre total d’opportunités et du montant des opportunités acquises par un programme marketing.

   ![](assets/image2015-7-21-16-3a21-3a4.png)

1. Influence marketing sur les opportunités fermées et gagnantes.

   Ce rapport montre le montant des recettes acquises et influencées par le marketing, par le nombre et le montant.

   ![](assets/image2015-7-21-16-3a41-3a55.png)

1. Opportunité fermée par source de piste

   Ce rapport présente toutes les opportunités fermées par source de piste et donne un aperçu clair des sources qui fonctionnent et de celles qui ne fonctionnent pas.

   ![](assets/image2015-7-21-10-3a34-3a50.png)

1. Durée de fermeture par source

   Ce rapport montre la relation entre le nombre moyen de jours pour fermer une opportunité et la source principale.

   ![](assets/image2015-7-21-10-3a35-3a3.png)

1. Ouverture de l’opportunité et de l’étape

   Ce rapport indique le nombre d’opportunités ouvertes à chaque étape du cycle des recettes.

   ![](assets/image2015-7-21-10-3a35-3a32.png)

1. Nombre d’opportunités par année par secteur industriel

   Ce rapport répond à la question : &quot;Est-ce que certaines industries offrent plus ou moins d&#39;opportunités d&#39;année en année ?&quot;

   ![](assets/image2015-7-21-10-3a35-3a45.png)

## Dimensions et mesures d’analyse des opportunités {#opportunity-analysis-dimensions-and-measures}

L’analyse des opportunités vous donne accès à toutes les dimensions liées aux prospects, aux entreprises et aux opportunités, ainsi qu’aux mesures liées aux opportunités. Utilisez ces dimensions d’analyse des opportunités et mesures pour répondre à des questions spécifiques dans votre rapport.

1. Attributs de la société

   | Dimension | Description |
   |---|---|
   | Chiffre d’affaires annuel | Chiffre d’affaires annuel de l’entreprise |
   | Ville | Ville où se trouve l’entreprise |
   | Pays | Le département où se trouve l’entreprise |
   | Secteur | L&#39;industrie dans laquelle se trouve l&#39;entreprise |
   | Nom de la société | Nom de la société |
   | Nombre d&#39;employés | Nombre d’employés de la société |
   | Code postal | Code postal de la société |
   | Code SIC | Code SIC de l’entreprise |
   | État | État où se trouve la société |

1. Attributs du lead

   | Dimension | Description |
   |---|---|
   | Figurant sur la liste de blocage | Le prospect est placé sur la liste bloquée |
   | Converti en opportunité | Le prospect est converti en opportunité |
   | E-mail non valide | Si le prospect possède une adresse électronique valide |
   | Marketing interrompu | L’avance est-elle suspendue des emails marketing ? |
   | Adresse e-mail | Adresse électronique du prospect |
   | Intitulé du poste | Fonction du responsable |
   | Nom complet | Nom complet de la piste |
   | Type source d’origine | Type de source d’origine du prospect |
   | Type de source d’enregistrement | Type de source enregistré du prospect |
   | Adresse e-mail du détenteur du lead | Adresse électronique du propriétaire du prospect |
   | Poste du détenteur du lead | Fonction du propriétaire du prospect |
   | Nom du propriétaire de piste | Nom du propriétaire du prospect |
   | Source du lead | Source de lead |
   | Statut du lead | Etat de piste |

1. Période créée de piste

   | Dimension | Description |
   |---|---|
   | Année de création de piste | L’année de création de la piste |
   | Trimestre de création de piste | Le trimestre au cours duquel la piste est créée |
   | Mois de création du lead | Mois de création de la piste |
   | Semaine de création du lead | La semaine de création de la piste |
   | Date de création du lead | Date de création de la piste |

1. Attributs d&#39;opportunité

   | Dimension | Description |
   |---|---|
   | Opportunité fermée | L&#39;opportunité est-elle fermée ? |
   | Catégorie de prévision de l’opportunité | Catégorie de prévisions d&#39;opportunité |
   | Nom de l&#39;opportunité | Nom de l’opportunité |
   | Étape de l’opportunité | Étape d’opportunité |
   | Type de l&#39;opportunité | Type d’opportunité |
   | Opportunité confirmée | Cette opportunité est-elle close et gagnée ? |
   | Opportunité influencée par le marketing | Cet indicateur indique si l’un des prospects/contacts a été acquis ou a réussi dans un programme marketing. Seuls les programmes pour lesquels un coût par période est défini sont pris en compte. |

1. Période de fermeture de l’opportunité

   | Dimension | Description |
   |---|---|
   | Année de fermeture de l&#39;opportunité | L&#39;année où l&#39;opportunité est fermée |
   | Trimestre de fermeture d’opportunité | Le trimestre au cours duquel l’opportunité est fermée |
   | Opportunité Fermée Mois | Le mois de fermeture de l’opportunité |
   | Semaine de fermeture de l&#39;opportunité | La semaine de fermeture de l’opportunité |
   | Date de fermeture de l&#39;opportunité | Date à laquelle l’opportunité est fermée |

1. Période de création de l’opportunité

   | Dimension | Description |
   |---|---|
   | Année de création de l&#39;opportunité | L&#39;année de création de l&#39;opportunité |
   | Opportunité créée par trimestre | Le trimestre au cours duquel l’opportunité est créée |
   | Mois de création de l’opportunité | Le mois de création de l&#39;opportunité |
   | Semaine de création de l’opportunité | La semaine de création de l&#39;opportunité |
   | Date de création d&#39;opportunité | Date de création de l&#39;opportunité |

1. Mesures

   | Mesurer | Description |
   |---|---|
   | Nombre moyen de jours pour fermer l’opportunité | Nombre moyen de jours pour fermer une opportunité |
   | Nombre moyen de jours pour fermer une opportunité (perdue) | Nombre moyen de jours jusqu’à une opportunité perdue |
   | Nombre moyen de jours pour fermer l’opportunité (gagnant) | Nombre moyen de jours jusqu’à une opportunité gagnée |
   | Nombre de toutes les opportunités | Nombre total de toutes les opportunités |
   | Nombre d’opportunités (fermé) | Nombre total d’opportunités qui sont fermées (gagnées ou perdues) |
   | Nombre d’opportunités (perdues) | Nombre total d’opportunités perdues |
   | Nombre d’opportunités (ouvertes) | Nombre total d’opportunités qui sont encore ouvertes |
   | Nombre d’opportunités (gagnant) | Nombre total d’opportunités gagnées |
   | Quantité d&#39;opportunités | Total du montant de l&#39;opportunité. Si plusieurs pistes sont associées à une opportunité, le montant de l’allocation est basé sur le score de piste. |
   | Montant de l&#39;opportunité (perdu) | Montant total des opportunités perdues. Si plusieurs pistes sont associées à une opportunité, le montant de l’allocation est basé sur le score de piste. |
   | Montant de l&#39;opportunité (ouvert) | Montant total pour les opportunités ouvertes. Si plusieurs pistes sont associées à une opportunité, le montant de l’allocation est basé sur le score de piste. |
   | Montant de l&#39;opportunité (gagné) | Montant total des opportunités gagnées. Si plusieurs pistes sont associées à une opportunité, le montant de l’allocation est basé sur le score de piste. |
