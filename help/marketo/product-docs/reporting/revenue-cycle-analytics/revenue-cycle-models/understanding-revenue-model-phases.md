# Comprendre les phases du modèle de chiffre d’affaires {#understanding-revenue-model-phases}

Les phases permettent de regrouper un certain nombre de phases. Parfois, plusieurs étapes d’un modèle reflètent une phase d’un entonnoir.

![—](assets/image2015-6-12-16-3a56-3a40.png)

## Définition des phases du modèle {#define-the-phases-of-the-model}

1. Cliquez sur **[!UICONTROL Phases]**.

   ![](assets/image2015-6-12-16-3a2-3a28.png)

1. Cliquez sur le bouton bleu pour faire glisser les phases vers le haut et vers le bas à travers les étapes.

   ![](assets/image2015-6-12-16-3a5-3a31.png)
