# Filtrage dans [!UICONTROL Informations sur les e-mails] {#filtering-in-email-insights}

Dans [!UICONTROL Insights sur les e-mails], il existe différentes manières de filtrer vos données.

## [!UICONTROL Email Insights] {#email-insights}

Cliquez sur le signe + pour commencer à ajouter des filtres. Ajoutez-en autant que vous le souhaitez.

![](assets/one-2.png)

Si vous avez plusieurs filtres dans la même catégorie, ils fusionnent et affichent le montant.

![](assets/state.png)

Si vous avez besoin de voir quels filtres vous avez appliqués, il vous suffit de cliquer dessus. Tu peux même en ajouter d&#39;autres.

![](assets/states.png)

Vous pouvez également ajouter d’autres types de filtres.

![](assets/os.png)

Et encore plus de filtres !

![](assets/more-filters.png)

Le graphique change après l’application de chaque filtre.

![](assets/filtered-chart.png)

Pour supprimer une catégorie, cliquez sur son X.

![](assets/filter1.png)

Pour effacer tous les filtres, cliquez sur le X à l’extrémité de la barre de filtres.

![](assets/filter2.png)

Pour les données spécifiques à la date, utilisez des périodes.

![](assets/date-click.png)

Effectuez un choix parmi des périodes prédéfinies courantes, sélectionnez des dates spécifiques et utilisez la période précédente pour la comparaison.

![](assets/date-range.png)

>[!NOTE]
>
>Les périodes s’appliquent à la page [!UICONTROL Analytics] et à la page [!UICONTROL Envois]. Toutefois, l’option **Comparer** ne s’applique qu’à la page [!UICONTROL Analytics].
