# Configuration initiale {#initial-setup}

Suivez les étapes ci-dessous pour configurer Dynamic Chat.

## Accès à Admin Console {#access-admin-console}

>[!NOTE]
>
>**Autorisations d’administrateur requises.**

1. Une fois que [!DNL Dynamic Chat] est activé pour votre instance Marketo, l’administrateur système désigné reçoit un e-mail de bienvenue. Dans cet e-mail, cliquez sur **[!UICONTROL Commencer]**.

   ![](assets/initial-setup-1.png)

1. Si vous avez déjà accédé à une application à l’aide d’une Adobe ID, vous accédez directement au Adobe Admin Console. Dans le cas contraire, [configurez votre Adobe ID](https://helpx.adobe.com/manage-account/using/create-update-adobe-id.html){target="_blank"}.

   ![](assets/initial-setup-2.png)

## Ajouter des utilisateurs {#add-users}

1. Après vous être connecté à Admin Console, ajoutez des utilisateurs et utilisatrices. Ce processus est [documenté ici](/help/marketo/product-docs/demand-generation/dynamic-chat/setup-and-configuration/add-or-remove-chat-users.md#add-a-chat-user){target="_blank"}.

Ensuite, il est temps de [connecter Dynamic Chat à Marketo](/help/marketo/product-docs/demand-generation/dynamic-chat/integrations/adobe-marketo-engage.md){target="_blank"}.
