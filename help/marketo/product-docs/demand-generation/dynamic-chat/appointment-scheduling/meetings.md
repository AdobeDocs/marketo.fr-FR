# Réunions {#meetings}

C&#39;est là que vous verrez toutes les nominations programmées par les visiteurs de votre site web à travers vos Dialogues. Vous trouverez ici l’adresse électronique de la personne ayant réservé le rendez-vous, l’agent auprès duquel elle a réservé le rendez-vous, le moment où le rendez-vous est prévu et si la réunion s’est déroulée ou non.
