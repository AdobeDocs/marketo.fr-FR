# Création d’un événement dans Marketo {#create-an-event-in-marketo}

Un événement Marketo effectue le suivi de la progression de vos utilisateurs par le biais d’un programme. Il transmet les informations d’enregistrement et extrait les informations de présence à l’aide de l’adaptateur ON24. L’événement capture le statut de vos personnes au fur et à mesure qu’elles passent à travers celui-ci.

Voici comment les données sont échangées :

![](assets/image2015-12-16-13-33-56.png)

Lorsque vous créez l’événement Marketo, sélectionnez **Webinaire** comme type de canal. Vous pouvez modifier ce canal dans Admin et en créer un nouveau. Si vous créez un canal, il doit être du type **Événement avec le webinaire** pour que l’intégration fonctionne. Voir [Création d’un canal de programme](/help/marketo/product-docs/administration/tags/create-a-program-channel.md){target=&quot;_blank&quot;} pour plus d’informations.

L’étape suivante consiste à [configuration des paramètres d’événement et synchronisation de Marketo avec votre webinaire](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/configure-event-settings-and-sync-marketo-with-your-webinar.md){target=&quot;_blank&quot;}.
