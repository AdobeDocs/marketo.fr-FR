# Exporter les données de campagne Web {#export-web-campaign-data}

Pour exporter vos données de campagne web, procédez comme suit.

1. Accédez à **Campagnes Web**.

   ![](assets/one-2.png)

1. Dans le coin supérieur droit de la page, cliquez sur l’icône Exporter CSV .

   ![](assets/two-2.png)

1. Ouvrez ou enregistrez votre fichier.

   ![](assets/three-2.png)

1. Affichez votre fichier pour consulter des statistiques utiles.

   ![](assets/four-1.png)
