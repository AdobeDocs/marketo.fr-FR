# Recherche de campagnes Web utilisant un segment spécifique {#find-web-campaigns-that-are-using-a-specific-segment}

Vous recherchez des campagnes web qui utilisent un segment spécifique ?

1. Accédez à **Segments**.

   ![](assets/new-dropdown-segments-hand-1.jpg)

1. Recherchez un **segment**. Sélectionnez le **nom du segment**. Dans le panneau de droite, cliquez sur **Campagnes associées** pour afficher les campagnes associées à ce segment spécifique.

   ![](assets/image2014-11-26-14-21-59.png)

1. Afficher les **Campagnes** associées au segment sélectionné.

   ![](assets/image2014-11-26-14-3a25-3a30.png)
