# Présentation de Web Personalization {#web-personalization-overview}

## Introduction {#introduction}

Pour les nouveaux clients Marketo, [!UICONTROL Real-Time Personalization] est divisé en quatre applications, accessibles à l’aide de ces deux mosaïques : [!UICONTROL Web Personalization] et [!UICONTROL ContentAI].

![](assets/pasted-image-at-2016-03-23-02-45-pm.png) ![](assets/mlm-homepage-content-ai-281-29.png)

>[!NOTE]
>
>[!UICONTROL Web Personalization] et [!UICONTROL ContentAI] sont disponibles dans toutes les langues prises en charge par Marketo (anglais, français, allemand, japonais, portugais, espagnol).

Les clients existants continuent à accéder aux applications par le biais de la mosaïque Personalization en temps réel jusqu’au renouvellement, après quoi ils utiliseront la méthode de connexion ci-dessus.

![](assets/image2016-2-9-8-3a52-3a32.png)

Lorsque vous cliquez sur la mosaïque [!UICONTROL Web Personalization], la page Tableau de bord s’affiche. Cliquez sur la mosaïque [!UICONTROL ContentAI] et accédez directement à la page Recommandations de contenu.

Les quatre applications sont les suivantes :

* [!UICONTROL Web Personalization]
* [!UICONTROL Marketing Web basé sur les comptes]
* [!UICONTROL Reciblage de site web]
* [!UICONTROL ContentAI]

La mosaïque **[!UICONTROL Web Personalization]** est votre point d’accès pour la création de campagnes web personnalisées. Marketing Web basé sur les comptes et reciblage Web. Vous pouvez ajouter du contenu sur la page Content Analytics.

**[!UICONTROL Marketing Web basé sur les comptes]** fonctionne avec Web Personalization et propose des comptes nommés. La page Comptes nommés fournit des informations graphiques sur les clients existants, le nombre total d’engagements et les tendances organisationnelles, ainsi qu’une liste des comptes. Vous pouvez également y créer des comptes nommés.

Le **[!UICONTROL reciblage de site web]** vous permet de revendre des audiences segmentées. Vous pouvez définir des segments, mais ne pouvez pas utiliser de comptes nommés dans le filtre.

**[!UICONTROL ContentAI]** est l’endroit où donner des recommandations de contenu. Vous pouvez ajouter facilement du contenu à la page de recommandations de contenu.

>[!NOTE]
>
>Pour accéder à [!UICONTROL Marketing web basé sur les comptes] ou [!UICONTROL Reciblage de site web], vous devez cliquer sur la mosaïque [!UICONTROL Personalization web], même si vous ne disposez pas de l’application [!UICONTROL Personalization web].

## Organisations {#organizations}

L’onglet Organisations affiche tous les détails (nom, lieu, activité et horodatage) des organisations qui ont visité le site Web au cours d’une période donnée. Le tableau peut être trié et organisé par heure, lieu, domaine et via une recherche de texte libre.

![](assets/image2014-11-10-19-3a23-3a18.png)

>[!TIP]
>
>« Les plus actifs » ou « Les plus récents » - Le tableau est organisé par organisations identifiées, puis par fournisseurs d’accès à Internet (indiqués par une icône de FAI). Il peut être trié en fonction des éléments suivants :
>
>* Les plus actives : les organisations les plus actives dans le tableau, en fonction du nombre de pages vues
>* Dernière : organisations les plus récentes dans le tableau (par défaut)
>* Pour obtenir des informations précieuses, filtrez en fonction des plus actives

**Partager des organisations**

Vous souhaitez partager les informations du visiteur de votre site web avec un collègue ? Cliquez sur une organisation, puis sur le lien Partager sur le panneau de droite.

Cette action active le recouvrement [!UICONTROL Partager cette organisation], où vous pouvez envoyer un e-mail partageant le nom et les détails de l’organisation qui a visité le site Web.

![](assets/image2014-11-10-19-3a25-3a42.png)

## Visiteurs {#visitors}

![](assets/wp-vis.jpg)

Cette page propose une analyse approfondie d’insight concernant les activités et le comportement en ligne de chaque visiteur. Les données du visiteur dans le tableau sont mises à jour dans les 24 à 48 heures.
