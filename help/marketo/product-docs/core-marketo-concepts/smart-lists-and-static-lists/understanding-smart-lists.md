# Compréhension des listes dynamiques {#understanding-smart-lists}

Les listes intelligentes vous permettent de trouver des groupes de personnes spécifiques en utilisant de simples filtres. Elles sont utilisées dans tout Marketo, par exemple dans les campagnes intelligentes, les programmes, les rapports, les segments, les modèles d’entreprise, les programmes d’engagement, le contenu dynamique, etc. Selon vos besoins, vous pouvez créer des listes dynamiques à deux endroits dans Marketo :

1. Dans le **Base**, où elles sont disponibles pour une utilisation partagée/globale (par exemple, &quot;Toutes les personnes désabonnées&quot;).
1. Dans **programmes** en tant qu’actifs locaux, où ils trouvent un groupe de personnes pertinentes pour ce programme spécifique (par exemple, &quot;Participé à un commerce&quot;).

Les principales étapes de création d’une liste dynamique sont les suivantes :

1. [Création d’une liste dynamique](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)
1. [Recherche et ajout de filtres à une liste dynamique](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md)
1. [Définition des filtres de liste dynamique](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/define-smart-list-filters.md)

Voici un exemple de ce à quoi ressemble une liste dynamique.

## Onglet Principal {#main-tab}

Ici, vous pouvez [modification du nom de la liste dynamique](/help/marketo/product-docs/core-marketo-concepts/miscellaneous/rename-a-marketo-asset.md) et voir des informations sur la liste dynamique.

![](assets/smartlist.png)

## Onglet Personnes {#people-tab}

C’est là que vous voyez vos résultats ; c’est-à-dire toutes les personnes qui remplissent les critères pour les filtres que vous avez [ajouté et défini](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md) sous l’onglet Liste dynamique .

![](assets/smartlist-people.png)

## Onglet Liste dynamique {#smart-list-tab}

Vous ajoutez et définissez des filtres pour afficher un groupe spécifique de personnes dans l’onglet Personnes .

![](assets/smartlist-filters.png)

OK, commençons !
