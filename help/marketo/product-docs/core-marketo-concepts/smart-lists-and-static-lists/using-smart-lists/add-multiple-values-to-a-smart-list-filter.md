# Ajout de plusieurs valeurs à un filtre de liste dynamique {#add-multiple-values-to-a-smart-list-filter}

>[!PREREQUISITES]
>
>* [Création d’une liste dynamique](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)
>* [Recherche et ajout de filtres à une liste dynamique](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md)


Disons que vous voulez trouver tout le monde en Californie, mais vous pouvez stocker les deux **Californie** et **CA** dans votre base de données. Pour inclure toutes les personnes applicables, vous pouvez utiliser deux **État** mais c&#39;est plus facile avec un filtre.

1. Accédez à **Activités marketing**.

   ![](assets/login-marketing-activities-2.png)

1. Recherchez et sélectionnez une liste dynamique, puis cliquez sur le bouton **Liste dynamique** .

   ![](assets/smarlist-choosefilters.png)

1. Cliquez sur le bouton **+** sur le filtre.

   ![](assets/smartlist-plussignhand-.png)

1. Vous pouvez choisir des valeurs à gauche ou simplement les saisir à droite, puis cliquer sur **OK**.

   ![](assets/image2014-9-11-17-3a51-3a39.png)

Un travail rapide !
