# Cloner une ressource dans un programme {#clone-an-asset-in-a-program}

Cloner un programme clone _tout_. Parfois, vous souhaitez simplement cloner une ressource. Voici comment faire.

>[!NOTE]
>
>Vous devez effectuer des étapes supplémentaires pour [cloner un groupe de test de page de destination](/help/marketo/product-docs/demand-generation/landing-pages/landing-page-actions/cloning-a-landing-page-test-group.md){target="_blank"}.

## Clonage d’une ressource locale {#clone-a-local-asset}

1. Accédez à **[!UICONTROL Activités marketing]**.

   ![](assets/login-marketing-activities.png)

1. Sélectionnez votre programme.

   ![](assets/image2014-9-23-15-3a56-3a12.png)

1. Cliquez avec le bouton droit sur la ressource locale à cloner. Cliquez sur **[!UICONTROL Cloner]**.

   ![](assets/image2014-9-23-15-3a56-3a25.png)

1. Chaque type de ressource présente une boîte de dialogue différente. Remplissez simplement les informations et cliquez sur **[!UICONTROL Cloner]**.

   ![](assets/image2014-9-23-15-3a56-3a34.png)

   >[!TIP]
   >
   >Vous pouvez également cloner une ressource vers un autre programme. Utilisez le menu déroulant **[!UICONTROL Programme]** pour effectuer votre sélection.

1. Fantastique ! Vous devriez maintenant voir la nouvelle ressource clonée.

   ![](assets/report.jpg)
