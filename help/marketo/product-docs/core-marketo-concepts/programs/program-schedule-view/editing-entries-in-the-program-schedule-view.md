# Modification d’entrées dans la vue Planning du programme {#editing-entries-in-the-program-schedule-view}

Vous pouvez apporter des modifications aux différents éléments de votre programme dans la vue de planning.

## Modifier le nom d&#39;une entrée {#edit-an-entrys-name}

1. Sélectionnez l’entrée à modifier.

   ![](assets/image2014-9-18-18-3a1-3a36.png)

1. Saisissez un nouveau nom et appuyez sur **[!UICONTROL Entrée/Retour]** sur le clavier pour confirmer la modification.

   ![](assets/image2014-9-18-18-3a1-3a53.png)

>[!CAUTION]
>
>Cela ne modifie que le nom d’affichage dans la vue de planning. Le nom de la ressource de votre programme reste le même.

## Modifier la description d&#39;une entrée {#edit-an-entrys-description}

1. Cliquez sur l’icône de description.

   ![](assets/image2014-9-18-18-3a3-3a7.png)

1. Modifiez votre description. Cliquez sur **[!UICONTROL Enregistrer]**

   ![](assets/image2014-9-18-18-3a3-3a22.png)

1. Fantastique ! Votre description a été modifiée.

   ![](assets/image2014-9-18-18-3a3-3a48.png)

## Modifier la date d&#39;une entrée {#edit-an-entrys-date}

1. Sélectionnez la nouvelle date.

   ![](assets/image2014-9-18-18-3a4-3a39.png)

Tu fais du rock ! La date de votre saisie a été déplacée vers une nouvelle date.

>[!NOTE]
>
> Les entrées de campagne intelligente et de programme de messagerie déjà exécutées ne peuvent pas être déplacées vers le passé.
