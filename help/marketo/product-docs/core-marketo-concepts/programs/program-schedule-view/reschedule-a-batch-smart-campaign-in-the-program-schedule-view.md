# Replanifier une campagne intelligente par lots dans la vue Planning du programme {#reschedule-a-batch-smart-campaign-in-the-program-schedule-view}

Vous souhaitez replanifier une exécution de campagne par lots ? Effectuez un glisser-déposer dans le futur dans la vue de planning du programme.

1. Sélectionnez la campagne intelligente à replanifier.

   ![](assets/image2015-5-19-12-3a8-3a28.png)

1. Effectuez un glisser-déposer vers la nouvelle date à laquelle vous souhaitez que la campagne s’exécute.

![](assets/image2015-5-19-12-3a12-3a1.png)

1. Fantastique ! La campagne intelligente a été replanifiée. L’heure peut également être modifiée à partir de la liste déroulante **[!UICONTROL Date]** dans les détails de l’entrée.

   ![](assets/image2015-5-19-12-3a15-3a38.png)
