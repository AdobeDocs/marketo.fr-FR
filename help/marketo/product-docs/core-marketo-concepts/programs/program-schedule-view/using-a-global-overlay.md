# Utilisation de la disposition d’affichage globale {#using-a-global-overlay}

La superposition globale dans la vue de planning du programme vous permet de voir votre programme par rapport aux autres ressources planifiées.

>[!PREREQUISITES]
>
>Pour utiliser cette fonctionnalité, vous devez disposer d’une licence [Calendrier marketing](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/understanding-the-calendar/issue-revoke-a-marketing-calendar-license.md){target="_blank"}.

## Utiliser la superposition globale {#use-the-global-overlay}

1. Sélectionnez votre programme.

   ![](assets/image2014-9-24-10-16-4.png)

1. Sélectionnez **[!UICONTROL Recouvrement]** dans le coin inférieur droit.

   ![](assets/image2014-9-24-10-3a16-3a9.png)

1. Les blocs pleins représentent les entrées à cette date. Cliquez pour afficher les détails.

   ![](assets/image2014-9-24-10-3a16-3a14.png)

   Les détails d’entrée des éléments de recouvrement sont en lecture seule. Cliquez sur le programme parent pour apporter des modifications.

   ![](assets/image2014-9-24-10-3a16-3a19.png)

## Utiliser un filtre enregistré comme recouvrement {#use-a-saved-filter-as-an-overlay}

Si vous avez [enregistré un filtre dans le calendrier marketing](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/working-with-the-calendar/saving-a-filter-definition-in-the-marketing-calendar.md){target="_blank"}, vous pouvez l’utiliser comme recouvrement dans la vue du planning du programme.

1. Cliquez sur le menu déroulant **[!UICONTROL Recouvrement]** et sélectionnez votre définition de filtre.

   ![](assets/image2014-9-24-10-3a16-3a26.png)

   Une superposition définie par le filtre que vous avez enregistré et sélectionné s’affiche maintenant.

   ![](assets/image2014-9-24-10-3a16-3a31.png)
