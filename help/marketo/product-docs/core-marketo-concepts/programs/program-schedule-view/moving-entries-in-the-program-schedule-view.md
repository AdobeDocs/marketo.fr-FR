# Déplacement des entrées dans la vue Planning du programme {#moving-entries-in-the-program-schedule-view}

Le déplacement d&#39;entrées dans la vue de planification les replanifie automatiquement.

>[!NOTE]
>
>Impossible de déplacer une campagne intelligente, un programme d’engagement ou une explosion d’e-mail déjà exécuté.

1. Sélectionnez votre entrée. Effectuez un glisser-déposer à une autre date.

   ![](assets/image2014-9-18-17-3a47-3a23.png)

1. Notre récupérateur de données annule ensuite l’approbation, modifie la date et approuve à nouveau la ressource. Il est cool comme ça.

   ![](assets/image2014-9-18-17-3a47-3a35.png)

   Votre saisie va maintenant être replanifiée.

   ![](assets/image2014-9-18-17-3a49-3a19.png)
