# Partage d’un Moment {#sharing-a-moment}

Vous souhaitez faire le point sur votre équipe ? Vous avez le choix entre deux options de partage.

>[!IMPORTANT]
>
>Le 2 octobre 2023, Adobe a supprimé l’application Marketo Moments de tous les magasins d’applications. Si l’application est déjà installée sur votre tablette ou votre appareil mobile, vous pouvez continuer à l’utiliser pour le moment. Une fois votre instance Marketo Engage migrée vers Adobe Identity pour l’authentification de Marketo, vous ne pourrez plus accéder à l’application. [En savoir plus](https://nation.marketo.com/t5/product-discussions/marketo-events-app-and-marketo-moments-app-end-of-life/m-p/340712/highlight/true#M193869){target="_blank"}.

1. Appuyez sur l’icône **[!UICONTROL Partager]** sur la carte.

   ![](assets/image2015-6-30-15-3a13-3a13.png)

1. Appuyez sur le menu de la carte, puis sur **[!UICONTROL Partager]** dans l’écran Détails.

   ![](assets/image2015-7-15-9-3a0-3a35.png)

   ![](assets/image2015-7-14-16-3a8-3a18.png)

1. Sélectionnez la manière dont vous souhaitez partager. Passons à l&#39;email.

   >[!NOTE]
   >
   >Le partage dépend des applications installées sur votre téléphone. Donc, si vous n&#39;avez pas [!DNL Facebook], vous ne le verrez pas dans Moments.

   ![](assets/image2015-7-14-16-3a11-3a42.png)

1. Saisissez l’adresse et l’objet de votre destinataire, puis appuyez sur **[!UICONTROL Envoyer]**.

   ![](assets/image2015-7-14-16-3a18-3a16.png)
