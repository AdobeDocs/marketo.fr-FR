# Modifier l&#39;étape dans le cycle de vente {#change-revenue-stage}

Si vous utilisez Revenue Cycle Modeler et que vous disposez d&#39;un modèle approuvé, vous pouvez décider de déplacer manuellement les personnes d&#39;une étape à une autre. Cette étape de flux vous aidera.

![](assets/change-revenue-stage-1.png)

1. Sélectionnez le **[!UICONTROL Modèle]**.

   ![](assets/change-revenue-stage-2.png)

1. Sélectionnez la **[!UICONTROL Nouvelle étape]** que vous souhaitez affecter et c’est terminé !

   ![](assets/change-revenue-stage-3.png)

   >[!CAUTION]
   >
   >L’entrepôt de données prête une attention particulière au moment où les personnes passent d’une étape à l’autre. Cela peut créer des données erronées si elles sont utilisées de manière incorrecte.
