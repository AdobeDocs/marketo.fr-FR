# Moment intéressant {#interesting-moment}

Si vous disposez de Marketo Sales Insight, vous pouvez utiliser l’étape de flux **moment intéressant** pour donner à votre équipe commerciale une visibilité sur les choses cool que font vos employés.

![](assets/interesting-moment-1.png)

1. Sélectionnez le moment intéressant **[!UICONTROL type]** que vous souhaitez utiliser.

   ![](assets/interesting-moment-2.png)

1. Créez une **[!UICONTROL description]** qui explique le moment intéressant pour votre équipe commerciale.

   ![](assets/interesting-moment-3.png)

   >[!TIP]
   >
   >_Moins est plus_. Collaborez avec votre équipe de vente pour vous assurer que les moments intéressants sont réellement intéressants.

Vous pouvez également utiliser des [jetons dans des moments intéressants](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/tabs-in-the-msi-panel/interesting-moments/trigger-tokens-for-interesting-moments.md){target="_blank"} pour créer des descriptions dynamiques vraiment utiles.
