# Activation d’une campagne dynamique de déclenchement | Onglet Planification {#activate-a-trigger-smart-campaign-schedule-tab}

L’activation d’une campagne intelligente de déclencheur est comme son activation. Voici ce qu&#39;il faut faire.

1. Dans le rapport **Planification** , cliquez sur **Activer**.

   ![](assets/activateprogram-hands.png)

   >[!TIP]
   >
   >Passez en revue la campagne dynamique avant de l’activer.

1. Cliquez sur **Activer** encore une fois.

   ![](assets/activatecampaign-hand.png)

   >[!CAUTION]
   >
   >Assurez-vous que la campagne est prête avant de cliquer sur Activer !

À partir de ce moment, toute personne qui se qualifie pour la liste dynamique passera par le flux défini par votre campagne dynamique.
