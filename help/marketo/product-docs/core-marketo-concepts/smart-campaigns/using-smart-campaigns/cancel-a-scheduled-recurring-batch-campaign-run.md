# Annulation d’une exécution de campagne par lots récurrente planifiée {#cancel-a-scheduled-recurring-batch-campaign-run}

Si vous avez une campagne par lots récurrente que vous ne souhaitez plus, vous pouvez annuler les exécutions futures. Voici comment.

1. Cliquez sur la campagne dynamique, puis sur **Planification** .

   ![](assets/image2014-9-22-16-3a44-3a51.png)

1. Cliquez sur **Périodicité du planning**.

   ![](assets/image2014-9-22-16-3a44-3a55.png)

   >[!TIP]
   >
   >Vous pouvez annuler une seule exécution en cliquant sur ![rouge x](assets/image2014-9-22-16-3a45-3a42.png) à côté. Découvrez comment [annuler l’exécution d’une campagne par lots planifiée ;](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/using-smart-campaigns/cancel-a-scheduled-batch-campaign-run.md).

1. Définir la planification sur **Aucun** puis **Enregistrer**.

   ![](assets/image2014-9-22-16-3a45-3a56.png)

   Voila ! Votre campagne dynamique ne s’exécutera plus.

   >[!CAUTION]
   >
   >Cela annule les exécutions futures, mais si une campagne dynamique est en cours d’exécution, vous ne pouvez pas l’annuler.
