# Renommer une ressource Marketo {#rename-a-marketo-asset}

Les dossiers, programmes et ressources locales peuvent tous être facilement renommés.

1. Sélectionnez une ressource que vous souhaitez renommer, puis cliquez sur le bloc de nom du panneau de droite.

   ![](assets/image2015-4-10-17-19-48.png)

1. Saisissez un nouveau nom dans le champ de texte. Cliquez sur **[!UICONTROL Enregistrer]**.

   ![](assets/image2015-4-10-17-3a19-3a33.png)

   >[!NOTE]
   >
   >Vous ne pouvez pas renommer les fichiers et images qui ont été téléchargés, ni une ressource qui est actuellement référencée dans une liste dynamique ou une campagne dynamique (dans les filtres, déclencheurs, étapes de flux, etc.).
