# Configuration de l’accès push Android pour l’application mobile {#configure-mobile-app-android-push-access}

1. Cliquez sur **Admin**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. Sélectionnez **Applications mobiles**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. Sélectionnez l’application mobile souhaitée.

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. Sous **Type d’accès push**, sélectionnez **Android** et cliquez sur **Configurer.**

   ![](assets/image2016-6-15-15-3a16-3a22.png)

   >[!NOTE]
   >
   >Vous aurez besoin d’une **clé API serveur** et d’un **numéro de projet** de la part du développeur de vos applications mobiles. Le développeur les reçoit en se connectant à Google Play Developer Console pour enregistrer votre application et activer la messagerie cloud.

1. Saisissez la clé API du serveur et le numéro de projet. Cliquez sur **Enregistrer**.

   ![](assets/image2015-4-22-18-3a54-3a54.png)

   Doux. Assurez-vous de configurer l’application avec iOS.
